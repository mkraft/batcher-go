# Batcher Go

Converts a heterogeneous or homogeneous stream into categorized batches.

## Install

```shell
$ go get github.com/mkraft/batcher-go
```

## Example

https://github.com/mkraft/wsproxy

## Tests

```shell
$ make test
$ make benchmark
```
